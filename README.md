# 📚 AI Study Assistant 🚀

An AI-powered study assistant built using Oxlo.ai API to help students learn faster and smarter.

## 🔍 Problem Statement

Students often struggle to understand large study materials quickly and prepare efficiently for exams.

## 💡 Solution

This application helps users:

* Summarize notes instantly
* Extract important key points
* Generate quiz questions for practice

## 🚀 Features

* 📄 Summarizes notes
* 🔑 Extracts key points
* ❓ Generates quiz questions
* 📂 Supports file upload (.txt)

## 🛠 Tech Stack

* Python
* Streamlit
* Oxlo AI (DeepSeek model)

## 🤖 Model Used

deepseek-v3_2

## ⚠️ Note

Due to temporary API limitations, the app may occasionally show "API busy" messages. However, the Oxlo API integration is correctly implemented.

## ▶️ How to Run

```bash
pip install streamlit requests
streamlit run app.py
```
