# 📚 AI Study Assistant

An AI-powered study assistant built using Oxlo.ai API.

## 🚀 Features
- Summarizes notes
- Extracts key points
- Generates quiz questions
- Supports file upload

## 🛠 Tech Stack
- Python
- Streamlit
- Oxlo AI (DeepSeek model)

## ▶️ How to Run
pip install -r requirements.txt
streamlit run app.py

## 🤖 Model Used
deepseek-v3_2
